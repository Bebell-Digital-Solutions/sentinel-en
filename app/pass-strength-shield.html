Security FAQs

Length is mathematically the most significant factor against brute-force attacks. Every additional character exponentially increases the number of possible combinations a hacker must guess. We recommend at least 12-16 characters.
Avoid common dictionary words. Attackers use "dictionary attacks" to guess common phrases instantly. Instead, use a passphrase made of random, unconnected words or a completely random string generated by a tool.
We highly recommend using a Password Manager. These tools generate and store strong, unique passwords for every site, so you only need to remember one "Master Password".
Contrary to old advice, you shouldn't change passwords arbitrarily if they are strong and unique. Only change them if you suspect a breach or if the service notifies you of a security incident.
